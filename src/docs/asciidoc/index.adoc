[[overview-http-verbs]]
== 용어 정리

API 에서 사용하는 공통된 용어는 다음과 같습니다.

|===
| 용어 | 설명

| `Admin`
| 관리자 권한을 가진 사용자를 지칭

| `Member`
| 이미 회원가입을 했고, 로그인을 한 상태로 서비스를 사용하는 사용자를 지칭

| `Anonymous`
| 로그인이 필요없는 익명 사용자를 지칭

|===
